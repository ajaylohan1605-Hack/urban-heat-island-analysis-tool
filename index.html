About

This tool analyzes Urban Heat Island using temperature differences and visualizes environmental impact.

UHI Calculation Theory

The Urban Heat Island (UHI) effect is calculated by measuring the temperature difference between urban and rural areas.

UHI = Turban − Trural

Methods:

  • Simple temperature difference method
  • Satellite-based Land Surface Temperature (LST)
  • Hotspot detection using mean + deviation

Steps:

  1. Collect temperature data
  2. Identify urban and rural regions
  3. Calculate surface temperature
  4. Compute temperature difference

This project uses a simplified model for demonstration.
